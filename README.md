# LGF Skool Automation

HighThrive admin dashboard for Skool member management, Zapier automation, and Supabase-backed data.

**Live site:** https://recillagimson.github.io/lgf-skool-automation/

## Quick deploy

```powershell
npm install
npm run gh:auth
npm run github:push
```

See [DEPLOY.md](./DEPLOY.md) for Supabase setup and GitHub Secrets.

## Local dev

```powershell
copy .env.example .env.local
npm run dev
```
